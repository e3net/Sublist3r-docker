# Sublist3r-docker
> Docker container of Sublist3r


## General info
[Sublist3r](https://github.com/aboul3la/Sublist3r) is a python tool designed to enumerate subdomains of websites using OSINT.


## Technologies
* Image use python 3

## Setup
1. Instal docker
2. Pull image

## Code Examples
    $ docker run -it --rm sublist3r:latest -d example.com
